DESCRIPTION
===========
  This example project shows how to use the IAR Embedded Workbench for ARM
 to develop code for Sonix SN32F707 Starter Kit V2. It shows basic use of
 I/O controller.
  It toggles LED1 and LED2 simultaneously.

COMPATIBILITY
=============
  The example project is compatible with Sonix SN32F707 Starter Kit V2.
  By default, the project is configured to use the J-Link SWD interface.

CONFIGURATION
=============
  The Workspace contains Getting Started example with few configurations.

  RAM Debug:   The example is loaded into internal RAM for SN32F707.
  Flash Debug: The example is flashed to embedded flash memory for SN32F707.

 Jumpers:
  J18 - Filled
  J19 - Filled

GETTING STARTED
===============

  1) Build and download the program.

  2) Run the program.
